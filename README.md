# Site Notes

**Site Notes** — это расширение для Firefox, которое позволяет сохранять заметки для сайтов прямо в браузере.

Для каждого сайта можно:
- сохранить заметку
- отредактировать её
- удалить заметку
- посмотреть список всех сохранённых заметок
- удалить заметку даже для другого сайта из общего списка

Все заметки хранятся **локально в браузере** через `storage.local`.

---

## Возможности

- заметки, привязанные к домену сайта
- автоматическое определение текущего сайта
- редактирование заметки для текущего сайта
- удаление заметки для текущего сайта
- просмотр всех заметок
- удаление заметок из общего списка
- минималистичный интерфейс
- без сервера
- без аккаунта
- без авторизации
